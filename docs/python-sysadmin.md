# Por qué un administrador de sistemas moderno necesita Python

## Bash vs Python

Bash es perfecto para tareas rápidas: mover archivos, ejecutar comandos, 
encadenar procesos. Pero cuando la complejidad aumenta, Bash se vuelve 
difícil de mantener y de leer.

Python entra cuando necesitamos:

- Procesar miles de líneas de logs
- Conectarnos a APIs externas
- Generar informes en Excel automáticamente
- Trabajar con datos estructurados (CSV, JSON)
- Escribir tests para verificar que nuestro código funciona

## Ventajas de Python para ASIR

1. **Legibilidad** — el código se entiende fácilmente
2. **Librerías** — Pandas, Requests, OpenPyXL hacen en una línea lo que en Bash serían 50
3. **Portabilidad** — funciona en Windows, Linux y Mac
4. **Automatización** — con schedule podemos ejecutar scripts automáticamente
5. **Orientación a objetos** — podemos modelar inventarios de red de forma estructurada

## Conclusión

Un administrador de sistemas que sabe Python puede automatizar auditorías 
de seguridad, detectar intrusos analizando logs y generar informes para 
gerencia sin intervención manual. Bash y Python no compiten — se complementan.
